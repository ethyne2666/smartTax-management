#  SmartTax  — Financial Intelligence Platform

SmartTax AI is a modern, AI-driven financial intelligence platform designed to help individuals, small businesses, and enterprises manage their finances, optimize taxes, and make data-driven investment decisions.

---

##  Overview

SmartTax combines financial analytics, visualization, and intelligent recommendations into a single platform. It provides users with:

- Real-time financial insights
- Tax optimization strategies
- Investment analysis
- AI-powered recommendations
- Clean and interactive dashboards

---

##  Key Features

###  Multi-Profile Support
- Individual users
- Small businesses
- Enterprises

Each profile has:
- Custom dashboards
- Tailored analytics
- Specific KPIs

---

###  Financial Dashboards

- Income vs Expenses tracking
- Net savings calculation
- Tax estimation
- Investment portfolio breakdown
- Financial health score

---

###  Data Visualization

Built with modern charting tools:
- Pie Charts (expenses, investments)
- Bar Charts (monthly data)
- Line Charts (trend analysis)

---

###  AI Recommendations

SmartTax provides:
- Expense reduction insights
- Tax-saving strategies
- Investment suggestions

All recommendations are dynamically generated based on user data.

---

###  Smart Search

- Overlay-based search UI
- Dynamic suggestions
- Route-based navigation
- Fast filtering logic

---

###  Insights & Content Pages

- Market insights
- Financial reports
- Economic trends
- Research-based content

---

###  Navigation System

- Responsive Navbar
- Mega menu (Insights)
- Sidebar dashboard navigation
- Smooth routing using React Router

---

##  Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS

### UI & Icons
- shadcn/ui
- lucide-react

### State & Data
- Local storage (no backend required)
